# Subscription gates the catalog, not installed use

Premium Access will be a subscription covering the entire Premium Design Pack catalog. When it expires, the Builder loses the ability to retrieve Premium Design Contracts, including Pack Releases retrieved previously; public Pack Previews remain visible, and each premium Design Contract already present in a Project remains usable there under a lasting Project License. This preserves offline installed use without DRM or runtime license checks while making continued catalog access and new Pack Releases the reason to renew.
