# Subscription gates the catalog, not installed use

Premium Access will be a subscription covering the entire Premium Design Pack catalog. When it expires, the Builder loses the ability to retrieve Premium Design Contracts, including Pack Releases retrieved previously; public Pack Previews remain visible, and each Premium Design Contract already installed in a Project keeps working there. Retrieval is gated by Premium Access, but once retrieved the `DESIGN.md` is a file the Builder keeps and may use in that Project — there are no per-Project entitlements, runtime checks, or DRM. This preserves offline installed use while making continued catalog access and new Pack Releases the reason to renew. See [0011](0011-retire-licensing-as-a-product-concept.md).
