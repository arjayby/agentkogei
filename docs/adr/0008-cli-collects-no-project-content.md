# CLI collects no Project content

The AgentKogei CLI will never send repository names, Git remotes, file contents, prompts, generated interfaces, or dependency lists to AgentKogei. Public Installation contacts only the selected Pack Source; premium operations send only the Builder identity, Pack Release, action type, and a random Project License identifier required for entitlements. Diagnostics and crash reporting will be opt-in, and Pack Evaluation will use AgentKogei-owned reference Projects. This limits behavioral analytics in exchange for repository privacy and trust.
