# Open software with commercial Design Packs

All product software—including the web application, Design Contract format, CLI, and evaluation tooling—will be open source so Builders can inspect, extend, and self-host it. Open Design Contracts will ship publicly, while Premium Design Contracts remain in private storage under a commercial license; self-hosting the software does not grant access to the premium catalog. An installed premium Design Contract may be visible inside a genuine public end-product Project without becoming openly licensed or eligible for extraction and redistribution. This keeps the product genuinely open while preserving its primary path to revenue.
