# Open software with commercial Design Packs

All product software—including the web application, Design Contract format, CLI, and evaluation tooling—will be open source so Builders can inspect, extend, and self-host it. Open Design Contracts will ship publicly, while Premium Design Contracts remain in private storage that self-hosting the software does not unlock. A Premium Design Contract already installed in a Project may be visible inside a genuine public end-product without becoming freely reusable elsewhere; the redistribution limits that apply live in the website Terms rather than in per-pack metadata. This keeps the product genuinely open while preserving its primary path to revenue. See [0011](0011-retire-licensing-as-a-product-concept.md).
