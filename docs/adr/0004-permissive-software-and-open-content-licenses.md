# Permissive software and open-content licenses

AgentKogei's web application, CLI, Design Contract format, and evaluation tooling will use the MIT License to maximize adoption and integration. AgentKogei-authored Open Design Contracts will use CC BY 4.0, while Premium Design Contracts will use a commercial Pack License implementing Project Licenses. We accept that released MIT software can be used in closed hosted forks because the business moat is the trusted Official Catalog, evaluated premium direction, brand, and release cadence rather than exclusive control of the product software.
