# Permissive software and open-content licenses

AgentKogei's web application, CLI, Design Pack specification, and validators will use the MIT License to maximize adoption and integration. AgentKogei-authored Open Design Pack prose and original visual resources will use CC BY 4.0, while Premium Design Packs will use a commercial Pack License implementing Project Licenses; every third-party pack must declare its own Pack License. We accept that released MIT software can be used in closed hosted forks because the business moat is the trusted catalog, evaluated premium content, brand, and release cadence rather than exclusive control of the platform code.
